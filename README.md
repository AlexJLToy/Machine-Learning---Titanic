# Titanic Survival Prediction — Machine Learning Exploration

An exploration of classical machine learning techniques applied to the [Kaggle Titanic dataset](https://www.kaggle.com/c/titanic), one of the most well-known introductory datasets in data science. This project compares multiple classification algorithms to predict passenger survival.

---

## 📂 Notebooks

| Notebook | Description |
|---|---|
| `Titanic Machine Learning - Logit.ipynb` | Logistic Regression baseline model |
| `Titanic Machine Learning - Decision Trees.ipynb` | Decision Tree classifier with tuning |
| `Titanic Machine Learning - Random Forest.ipynb` | Ensemble Random Forest model |
| `Titanic Machine Learning - Gradient Boosting.ipynb` | Gradient Boosting classifier |
| `Titanic Machine Learning - L1, L2, and Elastic Net Regularization.ipynb` | Regularization techniques applied to logistic regression |

---

## 🗂️ Dataset

This project uses the Titanic dataset from Kaggle, which contains information about passengers including age, sex, ticket class, fare, cabin, and whether they survived.

You can download the data directly from Kaggle: [Titanic - Machine Learning from Disaster](https://www.kaggle.com/c/titanic/data)

## 🛠️ Tech Stack

- **Python 3**
- **Jupyter Notebook**
- **pandas** — data manipulation
- **scikit-learn** — machine learning models
- **matplotlib / seaborn** — visualizations

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AlexJLToy/Machine-Learning---Titanic.git
   cd Machine-Learning---Titanic
   ```

2. Install dependencies:
   ```bash
   pip install pandas scikit-learn matplotlib seaborn jupyter
   ```

3. Download the Titanic dataset from Kaggle and place the CSV files in the project directory.

4. Launch Jupyter and open any notebook:
   ```bash
   jupyter notebook
   ```

---

## 📊 Models Covered

- **Logistic Regression** — interpretable baseline classifier
- **Decision Trees** — non-linear model with visual interpretability
- **Random Forest** — bagged ensemble to reduce overfitting
- **Gradient Boosting** — boosted ensemble for higher accuracy
- **Regularization (L1 / L2 / Elastic Net)** — techniques to improve generalization
